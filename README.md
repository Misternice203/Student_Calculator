### Student_Calculator
Simple student calculator to get average across all user subjects.

## Overview
This student grade calculator is a simple python program that allows users to enter student scores and calculate the total and average score. This project uses loops, vaiables, input validation. and basic arithmatic operations in Python.

## Features
- Accepts multiple student scores from user input
- Calculates the total score
- Calculates the average score
- Uses a for loop to process scores
- Includes basic input from user

## Technologies used
* Python
* Command line interface (CLI)

## How it works
1. The program asks the user to enter the student scores.
2. A loop processes each score.
3. The program updates a variable called total_score.
4. Ater all the scores are entered, The program calculates the average score.
